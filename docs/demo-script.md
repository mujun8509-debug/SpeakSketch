# SpeakSketch 演示脚本

## 最终 3 分钟演示脚本

### 0:00 - 0:20 打开页面

**操作**：打开前端页面 `http://127.0.0.1:5173/`。

**讲解**：
> "SpeakSketch 是一个纯语音控制的结构化绘图工具，可以把中文或英文自然语言指令转换成可编辑的 Fabric.js 图形。"

### 0:20 - 0:50 一键演示

**操作**：点击"一键演示"。

**讲解**：
> "先用一键演示快速展示完整绘图链路，包括复杂场景、空间关系、编辑操作和历史记录。"

### 0:50 - 1:20 中文 / 英文输入

**操作**：
1. 在调试输入中输入：`画一个红色圆形`
2. 在调试输入中输入：`draw a blue rectangle`

**讲解**：
> "系统支持中文指令和英文指令解析，都会统一转换成 DrawCommand 后执行。"

### 1:20 - 1:45 空间关系

**操作**：输入：`画一个男人站在树旁边`

**讲解**：
> "空间关系解析会识别主体、参照物和位置关系，并生成多个结构化绘图动作。"

### 1:45 - 2:15 历史与导出

**操作**：
1. 点击"撤销"
2. 点击"重做"
3. 点击"重放全部"
4. 点击"导出图片"

**讲解**：
> "绘图过程是结构化记录的，因此可以撤销、重做、重放，并导出 PNG 图片。"

### 2:15 - 2:40 可选 ASR 与 AI 风格化

**操作**：
1. 展示 ASR 设置区域。
2. 展示 AI 风格化区域。

**讲解**：
> "未配置真实后端 provider 时，浏览器识别、调试输入和 Mock 模式仍可使用。前端不保存任何讯飞或 Seedream API Key。"

### 2:40 - 3:00 后端 health 接口

**操作**：展示命令：

```bash
curl http://localhost:3001/api/health
```

**预期返回**：

```json
{"ok":true,"service":"SpeakSketch backend"}
```

**讲解**：
> "后端当前是安全代理骨架，提供 health、ASR mock 和 style-image mock 接口，真实 provider 会在后续独立 PR 中接入。"

## 演示路线

### 1. 打开项目并介绍

**操作**：打开浏览器访问项目地址

**讲解**：
> "欢迎使用 SpeakSketch，这是一个纯语音控制的结构化绘图工具。用户可以通过语音指令在画布上绘制各种图形，无需手动操作。"

### 2. 点击一键演示

**操作**：点击顶部"一键演示"按钮

**讲解**：
> "我们提供了一键演示功能，可以自动执行预设的指令序列，展示完整的绘图能力。"

**预期效果**：
- 自动清空画布
- 绘制海边场景（太阳、云、船、海平线、飞鸟）
- 绘制人物站在树旁边
- 绘制猫在汽车旁边
- 绘制鸟在天空中
- 修改图形颜色
- 放大最大图形

### 3. 展示复杂场景

**操作**：点击"画一个公园，有人、树、花、草地和小鸟"

**讲解**：
> "SpeakSketch 支持复杂场景的绘制，只需一句话即可生成包含多种元素的场景。"

**预期效果**：
- 画布上出现人物、树、花、草地和小鸟
- 各元素自动布局

### 4. 展示空间关系指令

**操作**：点击"画一只猫在汽车旁边"

**讲解**：
> "系统支持空间关系解析，可以理解'旁边'、'左边'、'右边'等位置关系描述。"

**预期效果**：
- 先绘制汽车
- 猫自动绘制在汽车旁边

### 5. 展示改色、放大、删除

**操作**：
1. 点击"把刚才那个图形改成蓝色"
2. 点击"把最大的图形放大一点"
3. 点击"删除最左边的图形"

**讲解**：
> "除了绘制，我们还支持各种编辑操作，包括改色、放大和删除。"

**预期效果**：
- 图形颜色变为蓝色
- 最大图形被放大
- 最左边的图形被删除

### 6. 展示撤销 / 重做

**操作**：
1. 点击"撤销"
2. 点击"重做"

**讲解**：
> "完整的历史管理功能，支持撤销和重做操作。"

**预期效果**：
- 撤销恢复上一步操作
- 重做恢复撤销的操作

### 7. 展示重放全部

**操作**：点击"重放全部"

**讲解**：
> "重放功能可以回顾整个创作过程，展示每一步的绘制操作。"

**预期效果**：
- 画布被清空
- 按顺序重新执行所有历史指令
- 语音提示"创作过程重放完成"

### 8. 展示导出图片

**操作**：点击工具栏"导出图片"按钮

**讲解**：
> "绘制完成后可以导出为 PNG 图片保存。"

**预期效果**：
- 画布内容被导出为图片
- 浏览器自动下载图片文件

### 9. 强调核心特点

**讲解**：
> "SpeakSketch 的核心价值在于：
> - **结构化**：所有图形都是可编辑的 Fabric.js 对象
> - **可编辑**：支持移动、改色、缩放、删除
> - **可撤销**：完整的历史管理
> - **可重放**：记录并重现创作过程"

## 常见问题应对

### Q：语音识别不工作？

**A**：
> "请确保使用 Chrome 浏览器，并允许麦克风权限。如果仍然无法使用，可以使用下方的调试输入框手动输入指令。"

### Q：图形绘制位置不对？

**A**：
> "系统会根据画布大小自动调整位置，如果需要精确控制，可以尝试使用位置表达，例如'在左上角画一个红色圆形'。"

### Q：如何清空画布？

**A**：
> "可以点击'清空画布'指令，或者重新点击一键演示自动清空后重新绘制。"

## 演示环境准备

### 浏览器要求
- Chrome 浏览器（推荐）
- 麦克风权限已开启

### 网络要求
- 核心绘图、调试输入和浏览器识别可本地运行
- 如需后端 Mock 代理接口，启动 `server/` 后端
- 如需真实云端 ASR 或 AI 图像生成，需要后续接入真实 provider 并配置后端环境变量

### 性能建议
- 避免同时绘制过多对象
- 复杂场景绘制后建议等待几秒让画面稳定
